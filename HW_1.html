How to View HTML Source?

Have you ever seen a Web page and wondered"Hey! How did they do that?"

View HTML Source Code:

Right-click in an HTML page and select "View Page Source"(in Chrome) or "View Source"(in Edge), or similar on other browsers.This will open a windows containing the HTML source code of the page.

Inspact an HTML Element:

Right-click in an element(or a blank area),and choose"Inspect" or "Inspact Element" to see what elementa are made up of (you will see both the HTML and the CSS).You can also edit the HTML or CSS on-fly in the Elements or Style panale that opens
